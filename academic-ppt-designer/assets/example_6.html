当前主流优化策略
运行时局部策略： 如 MagPy (状态监控)、BladeDISC (子图聚类)，侧重于局部的执行优化。
语义重构探索： 尝试利用 LLM 自动化解决底层限制，通过源码级重写突破动态语义屏障。
字节码拦截： TorchDynamo 成为行业共识，致力于实现 Eager 与 Graph 的透明切换。
核心技术瓶颈 (痛点)
静态分析受限： 传统模板难以覆盖复杂的 Python 嵌套、动态属性访问与外部库调用。
LLM 幻觉风险： 纯生成式方案缺乏底层语义约束知识，生成的优化代码稳定性极差。
系统性缺失： 现有研究多聚焦单一层面，缺少 “源码-AST-字节码” 的多粒度协同框架。
